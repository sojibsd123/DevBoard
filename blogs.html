Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which you can select elements in a DOM using selectors.

  • getElementById()
  • getElementsByTagName()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()
Question-2: What is the difference between innerHTML, innerText, and textContent ?

In JavaScript, "innerHTML" retrieves the complete HTML content within an element, including tags, while "innerText" only retrieves the visible text content of an element (ignoring HTML tags), and "textContent" retrieves all text content within an element, including hidden text, but without interpreting any HTML tags;

Question-3: What is event delegation in the DOM?

In the DOM, "event delegation" is a JavaScript design pattern where you attach a single event listener to a parent element instead of adding listeners to multiple child elements, allowing the parent to handle events that occur on its children by leveraging the "bubbling" phase of event propagation, where events travel up the DOM tree from the target element to its ancestors; this is particularly useful when dealing with a large number of similar child elements, optimizing performance by reducing the number of event listeners needed.

Question-4: What is event bubbling in the DOM?

Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

To add, remove, and modify HTML elements using JavaScript, you can use methods like createElement , appendChild , removeChild , and properties like innerHTML and textContent . For example, to add an element, you can create it using document. createElement and then append it to a parent element using appendChild .
